# Pro-Level Extension Assignments

## Overview

Pro-level assignments are **optional bonus work** for students who want to go further. Each assignment is worth **10 bonus points**, for a total of **50 bonus points** if all are completed.

These assignments extend your Hello World app with advanced features and production-ready implementations.

---

## Pro Assignment 1: Authentication System

**Points:** 10 bonus points

### Goal

Add user authentication to your application.

### Requirements

- User registration (sign up)
- User login
- Session management
- Protected routes/endpoints
- Logout functionality

### Technical Requirements

- Secure password storage (hashing)
- JWT tokens or session cookies
- Authentication middleware
- User database/storage

### Deliverables

- Working authentication system
- Documentation in README.md
- Security considerations documented

---

## Pro Assignment 2: Real-Time Features (WebSockets)

**Points:** 10 bonus points

### Goal

Add real-time functionality using WebSockets.

### Requirements

- WebSocket server setup
- Real-time message updates
- Client WebSocket connection
- Live updates without page refresh

### Technical Requirements

- WebSocket library (Socket.io for Node.js)
- Server WebSocket endpoint
- Client WebSocket connection
- Real-time event handling

### Deliverables

- Working real-time features
- Documentation in README.md
- Example use case demonstrated

---

## Pro Assignment 3: Database Integration

**Points:** 10 bonus points

### Goal

Add database integration to store and retrieve data.

### Requirements

- Database setup (SQLite, PostgreSQL, or MongoDB)
- Database schema/models
- CRUD operations (Create, Read, Update, Delete)
- Data persistence

### Technical Requirements

- Database connection
- ORM or query builder
- Migration system (optional)
- Data validation

### Deliverables

- Working database integration
- Database schema documented
- Example queries demonstrated

---

## Pro Assignment 4: Production Deployment

**Points:** 10 bonus points

### Goal

Deploy your application to a production environment.

### Requirements

- Deployed to production (Vercel, Railway, Heroku, etc.)
- Environment variables configured
- Production-ready configuration
- Live URL accessible

### Technical Requirements

- Deployment platform account
- Environment variables setup
- Production build configuration
- Domain/URL configuration

### Deliverables

- Live deployed application
- Deployment documentation
- Production URL shared

---

## Pro Assignment 5: Comprehensive Testing Suite

**Points:** 10 bonus points

### Goal

Add comprehensive testing to your application.

### Requirements

- Unit tests for server endpoints
- Integration tests for API
- Client-side tests (optional)
- Test coverage report

### Technical Requirements

- Testing framework (Jest or Vitest for Node.js)
- Test files organized
- Test coverage > 80%
- CI/CD integration (optional)

### Deliverables

- Complete test suite
- Test coverage report
- Testing documentation

## Submission Guidelines

### For Each Pro Assignment

1. **Implement the feature:**

   - Complete all requirements
   - Test thoroughly
   - Document in README.md

2. **Documentation:**

   - Feature description
   - How to use
   - Technical details
   - Screenshots/demos

3. **Commit and push:**
   ```bash
   git add .
   git commit -m "Pro Assignment [number]: [feature name]"
   git push
   ```

### Grading

Each pro assignment:

- Works as described: 5 points
- Code quality: 3 points
- Documentation: 2 points

**Total:** Up to 50 bonus points (10 points Ã— 5 assignments)
